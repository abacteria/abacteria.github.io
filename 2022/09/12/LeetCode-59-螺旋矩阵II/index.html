螺旋矩阵II

给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

方法：模拟

算法思路：

  1. 定义当前上下左右边界t,b,l,r，初始值num = 1，迭代终止值tar = n * n；
  2. 按顺序填入空矩阵，填完一条边时，边界要收缩。
  3. 以num <= tar作为循环条件，而不是l < r || t < b，以防n为奇数时，矩阵中心数字无法填充。
